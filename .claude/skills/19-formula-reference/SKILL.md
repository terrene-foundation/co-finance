# Formula Reference

Comprehensive formula reference sheets organized by finance course. Each sheet provides the formula, defines every variable, states the assumptions, and notes relationships between formulas. Designed as a quick-lookup resource during study sessions and (where permitted) as the basis for exam formula sheets.

## Quick Reference

1. **TVM Core**: PV = FV / (1 + r)^n -- the foundation of all finance
2. **CAPM**: E(r_i) = r_f + beta_i x [E(r_m) - r_f] -- pricing systematic risk
3. **WACC**: (E/V) x r_e + (D/V) x r_d x (1 - T) -- the firm's hurdle rate
4. **Black-Scholes**: C = S x N(d_1) - K x e^(-rT) x N(d_2) -- option pricing
5. **OLS Estimator**: beta_hat = (X'X)^(-1) x X'y -- the regression workhorse
