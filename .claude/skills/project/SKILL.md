# International Finance Classroom Materials

Frameworks, patterns, and templates for analyzing international finance events and creating pedagogically structured educational materials for university classrooms.

## Overview

This skill covers two interconnected domains:

1. **International Finance Analysis** — Systematic frameworks for identifying, analyzing, and synthesizing global financial events and connecting them to IF theory
2. **Educational Deep-Dive Creation** — Patterns for transforming analysis into structured, progressive learning materials ready for classroom use

## Reference Documentation

### Analysis Frameworks

- **[international-finance-analysis](international-finance-analysis.md)** — Analytical frameworks: impossible trinity mapping, oil shock transmission chains, double supply shock analysis, trade diversion patterns, dollar paradox analysis, country vulnerability assessment, and historical parallel methodology.

### Content Creation Patterns

- **[deep-dive-creation](deep-dive-creation.md)** — Document templates and quality standards: theoretical framework structure, transmission analysis pattern, country case study format, historical comparison format, discussion guide template, interconnection map design, and red team checklist.

## When to Use This Skill

- Starting a new International Finance events research project
- Creating deep-dive analysis of a specific discussion theme
- Designing discussion guides, debates, or group activities
- Analyzing how a global event affects different countries differently
- Connecting current events to international finance theory
- Building causal chain diagrams across interconnected events
