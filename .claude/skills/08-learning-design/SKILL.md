# Learning Design for Financial Education

Instructional design patterns, curriculum frameworks, assessment strategies, and gamification techniques tailored specifically for financial education.

## Overview

This skill covers pedagogical approaches for teaching finance:

- Bloom's taxonomy applied to financial learning objectives
- Curriculum design patterns and module sequencing
- Assessment types and rubric design for finance
- Gamification strategies including simulations and competitions

## Reference Documentation

### Learning Frameworks

- **[blooms-taxonomy](blooms-taxonomy.md)** - Bloom's taxonomy applied to financial education: Remember, Understand, Apply, Analyze, Evaluate, Create. Concrete examples at each level for finance topics.

### Curriculum Design

- **[curriculum-patterns](curriculum-patterns.md)** - Curriculum design patterns: spiral curriculum, mastery learning, project-based learning, case-study method. Financial education concept-to-analysis progression and module sequencing.

### Assessment

- **[assessment-design](assessment-design.md)** - Assessment types for financial education: concept checks, calculation exercises, case studies, portfolio projects, simulation exercises. Rubrics, difficulty calibration, formative vs summative.

### Engagement

- **[gamification](gamification.md)** - Gamification patterns: paper trading competitions, portfolio simulators, achievement badges, leaderboards, streak tracking, milestone celebrations. Implementation approaches.

## When to Use This Skill

Use this skill when:

- Designing a financial education course or module
- Writing learning objectives for finance content
- Creating quizzes, exercises, or projects for learners
- Sequencing financial topics from beginner to advanced
- Adding game-like elements to financial learning platforms
- Evaluating learner progress in financial literacy
