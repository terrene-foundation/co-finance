# Python Finance Libraries

Practical reference for the core Python libraries used in financial data analysis, computation, pricing, backtesting, and visualization.

## Overview

This skill covers the essential Python ecosystem for finance:

- pandas for financial time series manipulation and analysis
- numpy-financial for time-value-of-money and portfolio math
- backtrader for strategy backtesting and performance analysis
- QuantLib for instrument pricing and yield curve construction
- matplotlib, mplfinance, and plotly for financial visualization

## Reference Documentation

### Data Manipulation

- **[pandas-finance](pandas-finance.md)** - Financial time series with pandas: indexing, resampling, rolling windows, returns calculation, multi-ticker handling, and market data merging.

### Numerical Finance

- **[numpy-financial](numpy-financial.md)** - numpy-financial library for NPV, IRR, PMT, FV, PV. Vectorized portfolio math, covariance/correlation matrices, and Monte Carlo random number generation.

### Backtesting

- **[backtrader-guide](backtrader-guide.md)** - Backtrader framework: strategy classes, indicators, analyzers, observers, data feeds, cerebro engine, position sizing, order types, and commission models. Complete strategy example.

### Pricing and Modeling

- **[quantlib-basics](quantlib-basics.md)** - QuantLib Python: bond pricing, yield curve construction, interest rate models, options pricing (Black-Scholes, binomial, Monte Carlo), term structures, and day count conventions.

### Visualization

- **[visualization](visualization.md)** - Financial charts: candlestick charts (mplfinance), performance/equity curves, drawdown plots, correlation heatmaps, efficient frontier plots, and risk decomposition charts.

## When to Use This Skill

Use this skill when:

- Manipulating financial time series data
- Calculating returns, volatility, or risk metrics
- Building or analyzing backtested trading strategies
- Pricing bonds, options, or other derivatives
- Creating financial charts and dashboards
- Performing portfolio optimization calculations
