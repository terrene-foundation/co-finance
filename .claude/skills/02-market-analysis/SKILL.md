# Market Analysis

Comprehensive reference for technical and fundamental market analysis including indicators, chart patterns, financial ratios, and valuation methodologies with Python implementations.

## Overview

This skill covers the major approaches to analyzing securities:

- Technical indicators (moving averages, oscillators, volume-based)
- Chart patterns (reversal and continuation patterns)
- Fundamental ratios (valuation, profitability, leverage, liquidity)
- Valuation methods (DCF, comparables, DDM, residual income)

## Reference Documentation

### Technical Indicators

- **[technical-indicators](technical-indicators.md)** - Moving averages (SMA, EMA, WMA), RSI, MACD, Bollinger Bands, ATR, OBV, stochastic oscillator. Python implementations with pandas.

### Chart Patterns

- **[chart-patterns](chart-patterns.md)** - Support/resistance, head-and-shoulders, double top/bottom, triangles, flags, cup-and-handle. Pattern recognition concepts.

### Fundamental Ratios

- **[fundamental-ratios](fundamental-ratios.md)** - P/E, P/B, P/S, EV/EBITDA, ROE, ROA, debt-to-equity, current ratio, quick ratio, free cash flow yield. Calculation and interpretation.

### Valuation Methods

- **[valuation-methods](valuation-methods.md)** - DCF analysis, comparable company analysis, precedent transactions, dividend discount model, residual income model. Step-by-step implementations.

## When to Use This Skill

Use this skill when:

- Building stock screening or analysis tools
- Implementing trading signal generators
- Creating financial dashboards with ratio analysis
- Building valuation models for equity research
- Understanding market dynamics and price behavior
