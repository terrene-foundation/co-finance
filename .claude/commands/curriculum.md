## Bloom's Taxonomy for Finance

### Level 1 - Remember

**Verbs**: define, list, name, recall, identify
**Finance example**: "List the components of compound interest (principal, rate, time, frequency)."

### Level 2 - Understand

**Verbs**: explain, summarize, interpret, classify, compare
**Finance example**: "Explain why a bond's price moves inversely to interest rates."

### Level 3 - Apply

**Verbs**: calculate, solve, use, demonstrate, implement
**Finance example**: "Calculate the future value of a $500/month investment over 20 years at 7% annual return."

### Level 4 - Analyze

**Verbs**: compare, contrast, differentiate, examine, break down
**Finance example**: "Analyze the risk-return tradeoff between a 60/40 portfolio and a 90/10 portfolio."

### Level 5 - Evaluate

**Verbs**: assess, judge, critique, justify, recommend
**Finance example**: "Evaluate whether a given investment strategy is suitable for a retiree with moderate risk tolerance."

### Level 6 - Create

**Verbs**: design, construct, develop, formulate, compose
**Finance example**: "Design a diversified portfolio that targets 8% annual return with maximum 15% drawdown."

## Assessment Design Patterns

### Knowledge Check (Remember/Understand)

```
Multiple choice, matching, true/false
Example: "Which of the following best describes diversification?"
```

### Calculation Exercise (Apply)

```
Worked problems with specific inputs and expected outputs
Example: "Given a $10,000 investment at 6% compounded monthly for 5 years,
         calculate the ending balance. Show your work."
```

### Case Study (Analyze/Evaluate)

```
Present a scenario with data, ask for analysis and recommendation
Example: "Review this client's portfolio allocation and current life stage.
         Identify three specific risks and recommend adjustments."
```

### Portfolio Project (Create)

```
Open-ended construction with constraints and rubric
Example: "Build a portfolio of 5-10 ETFs that meets these criteria:
         - Target annual return: 7-9%
         - Maximum single-sector exposure: 30%
         - Must include international diversification
         Justify each selection."
```

## Curriculum Sequencing Pattern

```
Module 1: Financial Foundations (Remember/Understand)
  └── Time value of money, compound interest, inflation
Module 2: Investment Basics (Understand/Apply)
  └── Asset classes, risk-return relationship, reading charts
Module 3: Portfolio Construction (Apply/Analyze)
  └── Diversification, correlation, rebalancing
Module 4: Strategy Development (Analyze/Evaluate)
  └── Factor investing, market regimes, strategy comparison
Module 5: Capstone Project (Evaluate/Create)
  └── Build and defend a complete investment plan
```
